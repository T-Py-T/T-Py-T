> **Currently running AI model serving on GPU worker nodes in my home lab.**

My current work focuses on **serving AI models at home** — standing up GPU worker nodes and scheduling model serving and MLOps workloads onto them with Kubernetes. This involves:

- **GPU worker nodes** running inference and training jobs on Kubernetes
- **Model serving** with KServe, Ray Serve, and vLLM for LLMs and custom models
- **GPU scheduling** via the NVIDIA device plugin, with time-slicing for shared GPUs
- **Pipeline automation** from model registry to deployment
- **AI-augmented tooling** wired into everyday developer and ops workflows

**Cluster nodes** joined to the K8s cluster:

- **Mac Studio** — Apple Silicon node for local inference & dev workloads
- **DGX Spark** — NVIDIA Grace Blackwell node for GPU model serving & training
